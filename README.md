# ☕ Seaside Memory Café


A pixel-art café management browser game. Play as a cat waiter delivering food to customers before they lose patience — all in a cozy seaside café setting.

---

## 🎮 How to Play

| Action | Control |
|--------|---------|
| Move | `W A S D` or arrow keys |
| Pick up order | Walk to the counter when food is ready |
| Deliver food | Walk to the correct table |
| Return food | Press `R` while carrying |
| Chase the duck | Click on it 3 times |

### Game Flow
1. **Seon** (the cook) prepares orders — but every 4th dish might be wrong
2. Walk to the **counter** to pick up the order and check the receipt
3. Deliver the food to the **correct table** before the customer loses patience
4. If the dish is wrong, click **Again** on the receipt to remake it (3s delay)
5. Earn points for correct deliveries — wrong orders cost **-10**
6. Customers leaving without food cost **-5**
7. Reach **250 points** before the **3-minute timer** runs out to win!

### Watch Out
- 🦆 The **duck** bounces around and grows larger if ignored — click it away!
- While the duck is visible, all customers get impatient **20% faster**

| Sprite | Name | Role |
|--------|------|------|
| 🐱 | **You (the Cat)** | Player character — delivers food |
| 👨‍🍳 | **Seon** | The cook — prepares orders |
| 🍶 | **Customers** | Wait at tables — serve them in time |
| 🦆 | **The Duck** | Chaos agent — click to chase away |

## 🚀 Running the Game

No installation needed. Just open `index.html` in any modern browser.

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/seasidememorycafe.git

# Open in browser
open index.html
```
